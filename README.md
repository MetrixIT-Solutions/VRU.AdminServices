# Swit.AdminApi
V Reserve U (We Reserve You) - Admin API service for Admin Web Application.


## Getting started

Clone an existing Git repository with the following command:

1. cd ../workspace/VRU/Backend/
2. Clone the repo using following git command: `git clone https://github.com/MetrixIT-Solutions/VRU.AdminServices.git`
3. Go to the appropriate directory to start the Api service: `cd VRU.AdminServices`
4. Required Node Version >=18.12.1; Recommonded 18.20.5
5. Run `yarn` or `npm install` to install dependencies
6. Run the command: `npm run start`
7. You can find the sevice status on browser with the url `http://localhost:3517`

Push an existing Git repository with the following command:

```
cd existing_repo
git remote add origin git clone https://github.com/MetrixIT-Solutions/VRU.AdminServices.git
git branch -M main
git push -uf origin main
```
